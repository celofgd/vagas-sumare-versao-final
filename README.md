# Vagas em Sumaré

Sistema simples de divulgação de vagas de emprego para fins acadêmicos. Desenvolvido em **PHP** com **MySQL** e **HTML/CSS/JavaScript**.

---

## Como rodar

1. Coloque o projeto em um servidor com PHP e MySQL (por exemplo XAMPP).
2. Crie o banco executando o `schema.sql`.
3. Ajuste usuário e senha do banco em `config_banco.php` se necessário.
4. Acesse pelo navegador (ex.: `http://localhost/site/`).
